# Hextech LoL Bridge

一个基于 **WinUI 3 + WebView2** 的 **《英雄联盟》× 键盘鼠标灯光联动工具**。

项目目标是读取《英雄联盟》本地状态，并把技能、召唤师技能、血量、蓝量、死亡倒计时、英雄主题色等信息映射为 **键盘与鼠标灯效**。

> 当前项目仍处于 **Alpha 测试阶段**。
>
> 目前已实现的设备支持以 **罗技 Logitech / LIGHTSYNC** 为主，其他品牌暂未适配。

---

## 项目简介

Hextech LoL Bridge 是一个面向 Windows 的桌面工具，尝试把《英雄联盟》的局内状态与 Logitech LIGHTSYNC 灯效连接起来。

当前项目已经围绕以下方向进行了实现与实验：

- 英雄主题色
- 召唤师技能主题色
- 技能键位映射
- 血量条 / 蓝量条
- 死亡倒计时
- 鼠标主题色联动
- PRO X TKL 键盘布局适配

## 当前支持情况

### 已支持

- Windows 桌面环境
- 键盘灯光联动
- 鼠标灯光联动
- Logitech / LIGHTSYNC 设备
- WinUI 3 + WebView2 图形界面

## 功能概览

### 1. 召唤师技能主题色

支持为不同召唤师技能设置颜色，例如：

- 闪现
- 传送
- 疾跑
- 点燃
- 治疗
- 惩戒
- 屏障
- 虚弱
- 净化
- 清晰术
- 雪球 / 标记
- 海克斯闪现（扩展）

支持：

- 色轮选色
- 十六进制颜色输入
- 恢复默认
- 测试灯光

### 2. 英雄主题色

支持为英雄单独设置主题色，并用于：

- 外围按键主题色
- 鼠标主题色
- 局内技能统一主题色

### 3. 键盘灯效逻辑

当前已尝试实现：

- Q / W / E / R 技能状态灯
- D / F 召唤师技能灯
- 血量条
- 蓝量条
- 死亡倒计时
- 英雄主题色外围灯效

### 4. 鼠标灯效逻辑

当前逻辑：

- 非对局阶段：控制权交还 G HUB
- 对局阶段：鼠标主题色联动
